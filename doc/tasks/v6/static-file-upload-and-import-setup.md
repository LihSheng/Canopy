# Static File Upload And Import Setup

## Goal

Let a user upload a static file, inspect it, and start the import flow from the dataset workspace path.

## Tasks

- [ ] Accept static-file uploads from the setup screen.
- [ ] Save the uploaded file and capture basic file metadata.
- [ ] Profile the uploaded workbook or file source object.
- [ ] Let the user select sheets or source objects to create datasets.
- [ ] Create the connection record for the import session.

## Testing

- [ ] Verify upload acceptance for the supported file types.
- [ ] Verify preview/profile generation after upload.
- [ ] Verify dataset selection from the detected source objects.
